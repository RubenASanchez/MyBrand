Web Application


I developed a mobile app for generating tree structures. These tree structures are usefull for visualizing syntactic relations in linguistics and computer science. I felt that the available tools for generating such structures were cumbersome and many had a steep learning curve especailly for those not familiar with programing. I set out to create an app that was for the student intuitive and available to them regardless of the operating system of their mobile devices. This provided the students with more time to spend on interacting with the lesson instead of struggling through a proprietory interface.
